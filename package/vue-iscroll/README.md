# vue-iscroll-aj

vue-iscroll-aj 是基于 iscroll.js 的 Vue组件

## Install

```sh
$ npm install vue-iscroll-aj --save
```

## Usage
```sh
import VueIscroll from 'vue-iscroll-aj'
  export default {
    name: 'app',
    data () {
      return {
        n: 40
      }
    },
    components: {VueIscroll}
  }
</script>
```
